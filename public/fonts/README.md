# Intro font: Aktiv Grotesk Extended

The three intro screens use **Aktiv Grotesk Extended (Semibold)**.

To load the font on all devices, add the webfont files here:

- `AktivGroteskExtended-SemiBold.woff2` (preferred)
- `AktivGroteskExtended-SemiBold.woff` (fallback)

If the files are missing, the site falls back to the system font. If you have Aktiv Grotesk Extended installed on your machine, it will be used via `local()`.
